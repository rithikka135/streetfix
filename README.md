# StreetFix – Smart Civic Issue Reporting System

StreetFix is a web-based civic engagement platform designed to help citizens report and track local street-related issues efficiently. The application allows users to report problems such as potholes, garbage accumulation, water leakage, damaged streetlights, and road maintenance issues by providing location details and issue descriptions.

The platform improves communication between citizens and local authorities by enabling real-time issue submission, transparent tracking, and faster response management.

### Key Features

* **Issue Reporting System** – Submit complaints with detailed descriptions
* **Location-Based Tracking** – Identify exact issue location
* **Category Selection** – Report potholes, garbage, drainage issues, streetlight failures, etc.
* **Status Monitoring** – Track complaint progress and resolution
* **User-Friendly Interface** – Clean and responsive design
* **Real-Time Updates** – Improved visibility of reported issues

### Technologies Used

* **Programming Language:** Java / Kotlin
* **Development Environment:** Android Studio
* **UI Design:** XML Layouts
* **Version Control:** Git & [GitHub]




### Purpose

The goal of StreetFix is to bridge the gap between citizens and municipal authorities by creating a transparent and efficient digital reporting system for public infrastructure maintenance.
